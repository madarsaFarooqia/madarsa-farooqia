format: DUF-{TYPE}-{INSTITUTE}-{YYYYMMDD}-{SEQ}
examples:  
 DUF-ZKT-MDR-20260420-000000123
DUF-SDQ-MSJ-20260420-000000045
DUF-FTR-NSW-20260420-000000012
DUF-LIL-MDR-20260420-000000078

1: Prefix:
DUF = Darul Uloom Farooqia

2: Donation Type Codes
ZKT = Zakat
SDQ = Sadaqah
FTR = Fitra (Zakat al-Fitr)
LIL = Lillah
WFQ = Waqf (Endowment)
GEN = General Donation (optional fallback)

3. Institute Codes
   MDR = Madarsa (Darul Uloom Farooqia main education system)
   MSJ = Masjid
   NSW = Niswaan (Women’s section)
